- The task is to set a purchase limit to the customer,

- Add a new boolean field “Activate Purchase Limit” in Point of Sale->Orders->Customers->Sales & Purchase(Page)->Purchase Limit, if it is activated the user can set a purchase limit amount for that customer.

- When doing payment in PoS, if no customer is selected a Popup will appear which says “Select a Customer ”

- When doing payment in PoS, if the customer is selected and if he is having a purchase limit and the purchase done by the customer exceeds the limit, a Popup will appear with a message showing his purchase limit amount.
